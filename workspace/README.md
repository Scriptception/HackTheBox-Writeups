# workspace/

Scratch area for active boxes. Everything in here is gitignored.

Use one folder per box while it's in progress:

```
workspace/
  <BoxName>/
    nmap/
    loot/
    notes.md
    exploit.py
```

Once a box is done, lift the polished writeup into the proper track folder
(e.g. `Starting-Point/Tier-1/<BoxName>/writeup.md`) and clear the workspace
folder at your leisure. Nothing in here gets committed.
